# Longitude Geo Intelligence — V69.2 Modular

Esta versão inicia a modularização real sem remover as funções existentes.

## Instalação na pasta oficial

Execute `ATUALIZAR_V69_2_MODULAR.bat` como administrador.

Depois:

```cmd
cd /d D:\COMPARTILHAMENTO\longitude-geo-mvp-v14\longitude-geo-mvp-v14
npm run dev
```

Abra `http://localhost:5173` e procure o menu **Cloud Importer / Consulta UTM**.
